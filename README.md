# Mini RBAC

`Python(FastAPI)`、`VUE3`实现，仅保留核心权限控制的极简后台管理。

## 功能

### 权限控制

- [x] 前端菜单权限控制
- [x] 前端路由权限控制
- [x] 前端按钮权限控制
- [x] 后端接口权限控制

### 业务功能

- [x] 登录、退出、切换角色
- [x] 用户管理 -> 新增、编辑、删除、查询
- [x] 角色管理 -> 新增、编辑、删除、查询
- [x] 菜单管理 -> 新增、编辑、删除、查询
